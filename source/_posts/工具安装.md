安装 [brew](https://brew.sh/)
```
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```
安装 sequel mysql客户端
```
$ brew cask install sequel-pro
```
安装 rds redis客户端
```
$ brew cask install rdm
```
